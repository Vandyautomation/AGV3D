# AGV3D
Projek AGV untuk industri. Projek ini diharapkan dapat membantu mensimulasikan gerak AGV 3D menggunakan website Next.js.

## Getting Started

Jalankan development server:

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```

Buka http://localhost:3000 di browser.

Untuk mulai mengedit, ubah `app/page.tsx`. Perubahan akan tampil otomatis.
